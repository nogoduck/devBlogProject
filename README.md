## 개발 인원
- 개인 프로젝트

## 주요 기능
- **회원가입**
	1. 중복된 이메일인지 조회
	2. 비밀번호 유효성 검사
- **로그인**
	1. 이메일 유효성 검사 
	2. 가입된 계정인지 확인
- **계획 및 투두**
	1. 카테고리 추가
	2. 메모 추가, 삭제, 수정, 완료
- **게시판**
	1. 글 쓰기, 수정
	2. 댓글
	3. 좋아요
- **설정**
	1. 닉네임 변경
	2. 프로필 사진 변경(이미지 자르기 가능), 제거
	3. 비밀번호 변경
	4. 회원탈퇴
